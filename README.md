# ISYHelperHABridge
Python interface between the UDI ISY and HABridge


# Installation

cd /home/pi
mkdir ihb
cd ibh
git clone https://github.com/jimboca/ISYHelperHABridge

cd ISYHelperHABridge
./install_habridge.sh

Configure the habridge by going to add a Harmony Hub
http://your_pi_ip_address

./install_ihb.sh
